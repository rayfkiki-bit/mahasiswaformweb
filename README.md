# Aplikasi CRUD Mahasiswa

Proyek ini adalah aplikasi manajemen data mahasiswa yang menghubungkan Frontend Next.js dengan Backend Express.js dan Database.

## Teknologi yang Digunakan
- **Frontend:** Next.js, Tailwind CSS, TypeScript
- **Backend:** Express.js, TypeScript, MySQL
- **Database:** MySQL
